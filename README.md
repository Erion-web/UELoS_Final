# UELoS - University Equipment Loan System

A Node.js application demonstrating a loan management system for university equipment.

## Setup

```bash
npm install
npm start
```

## Structure

- `src/domain/` - Domain model classes
- `src/repo/` - Repository implementations
- `src/strategy/` - Strategy pattern implementations
- `src/services/` - Business logic services
- `src/controller/` - Controllers for handling requests
- `src/infra/` - Infrastructure adapters and factories
